# Ritual - サポート

ルーティンを「順番どおり」に実行するアプリ。

## お問い合わせ

アプリに関するご質問・不具合報告・ご要望は、以下よりお気軽にお寄せください。

- **GitHub Issues**: [こちらから報告](https://github.com/tekarimeron11/ritual-support/issues)

## よくある質問

### Proにアップグレードすると何ができますか？
無料版ではルーティンを3つまで作成できます。Proにアップグレードすると、ルーティンを無制限に作成できます。買い切りのため、追加課金はありません。

### ストリークが途切れる条件は？
2-Day Ruleを採用しています。1日サボってもストリークは途切れません。2日連続でスキップするとリセットされます。

### Apple Watchでも使えますか？
はい。Apple Watch単体でルーティンのステップ実行ができます。

### データのバックアップはできますか？
設定画面からJSON形式でエクスポート/インポートが可能です。Pro版ではiCloud同期にも対応しています。

## プライバシーポリシー

Ritualは、ユーザーの個人情報を収集しません。

- アプリのデータはデバイス上（およびiCloud同期を有効にした場合はiCloud上）にのみ保存されます
- 第三者へのデータ送信は一切行いません
- アナリティクスツールやトラッキングSDKは使用していません
